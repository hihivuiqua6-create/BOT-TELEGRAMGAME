CÁCH DÙNG:
1) Upload backup.php lên InfinityFree.
2) Trong web admin bot > Cài đặt:
   Domain Infinity đã up backup.php = https://domain-cua-ban.com
   hoặc https://domain-cua-ban.com/backup.php
3) Vào tab Backup bấm Đẩy backup ngay.
4) Khi Render restart, bot tự kéo backup mới nhất.

Không dùng secret theo yêu cầu.
