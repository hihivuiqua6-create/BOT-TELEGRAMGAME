CÁCH DÙNG BACKUP PHP INFINITY:
1) Upload backup.php lên InfinityFree.
2) Mở backup.php sửa $SECRET giống với Backup Secret trong web admin Python.
3) Trong web admin Python > Cài đặt:
   Backup URL PHP = https://domain-cua-ban/backup.php
   Backup Secret = secret giống trong backup.php
4) Bot sẽ tự kéo backup khi start và tự đẩy backup định kỳ.
